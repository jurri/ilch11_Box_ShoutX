""""""""""""""""""""""""""""""""""

Beschreibung:
-------------
gfa_shoutbox ist eine Weiterentwicklung der normalen ilch-Shoutbox so dass man da auch bbcode verwenden kann und Gäste nicht mehr unerkannt posten können.


Entwickelt
----------
° FeTTsack
° auf Basis von IlchClan 1.1 p


Installation:
-------------
° Backup machen
° alle Dateien im Ordner upload, in ihrer Ordnerstrucktur hochladen


Bekannte Einschränkungen / Fehler:
----------------------------------
° noch keine bekannt.

Log
----------------------------------
° v1.0
- anzeige ob Gastpost oder nicht
- bbcode funktion
- ip des Users wird abgelegt
- nutzt die Standard Shoutbox-Datenbank
